## Setup

```bash
npm install
```

Create `backend/.env`:

```
SUPABASE_URL=your_supabase_url
SUPABASE_SERVICE_KEY=your_service_key
```

## Development

Run both frontend and backend:

```bash
npm run dev
```

Run separately:

```bash
npm run dev:frontend   # http://localhost:3000
npm run dev:backend    # http://localhost:3001
```

## Build

```bash
npm run build
```
